## Description
Rock Paper Scissors is a two-player hand game usually played between two people. The game has three possible outcomes: a draw, a win for the player, or a win for the computer. This project brings the classic game to the web browser, allowing users to play against a computer opponent.

## Features
- User-friendly interface
- Interactive gameplay
- Real-time scoring
- Reset option to start a new game

## How to Play
1. Open the game in a web browser.
2. Click on one of the three choices: Rock, Paper, or Scissors.
3. The computer will randomly select its choice.
4. The result of the round will be displayed, along with the updated scores.
5. Continue playing until you decide to reset the game.

## Dependencies
No external libraries or frameworks are used in this project. Only basic HTML, CSS, and JavaScript are required.

## Usage
1. Open the `index.html` file in a web browser.
2. Click on your desired choice (Rock, Paper, or Scissors).
3. Enjoy playing the game!
